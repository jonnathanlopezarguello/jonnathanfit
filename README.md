# JONNATHAN FIT

Diario personal de entrenamiento y nutrición basado en evidencia. Aplicación web de un solo archivo (HTML + CSS + JS, sin dependencias ni build), instalable como PWA y con los datos guardados en el propio navegador.

## Características
- **Nutrición**: objetivo de calorías y macros (proteína, carbos, grasa y **fibra**) por fórmula Mifflin-St Jeor; ~130 alimentos, alimentos propios y recetas en un toque, edición de cantidades.
- **Entreno**: división adaptada a tus días (3/4/5/6), **sobrecarga progresiva** (última marca + objetivo de hoy), **récords y 1RM estimado**, y volumen semanal por grupo muscular con aviso.
- **Progreso**: peso corporal, **cintura/abdomen**, resumen semanal y **ajuste automático de calorías** según tu tendencia.
- **Suplementos** con evidencia, registro de actividad/pasos, modo claro/oscuro y copia de seguridad (exportar/importar).

## Uso
Abre `index.html` en el navegador, o visita la versión publicada en GitHub Pages.

## Aviso
Herramienta orientativa y educativa. No sustituye el consejo de un médico o dietista-nutricionista.
